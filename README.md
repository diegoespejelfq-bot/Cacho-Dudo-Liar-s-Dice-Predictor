# Cacho-Dudo-Liar-s-Dice-Predictor
Cacho es un juego de dados donde cada jugador ve solo los suyos y apuesta cuántas veces aparece un número entre todos. El siguiente debe subir la apuesta o dudar. Si duda y la apuesta es falsa, gana; si es verdadera, pierde. Puede decir “cacho” si cree que es exacta. El 1 es comodín y el objetivo es quedar con al menos un dado.
